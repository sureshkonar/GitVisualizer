## LinkedIn Post Draft

For the longest time, Git felt like this:

You run a command.  
Something changes.  
You hope it was the right thing.

I wanted to change that feeling.

So I built **GitVisualizer**: an interactive Git learning playground where commands are not just text, they are **visible actions**.

You can:

- run real Git-style commands in a live sandbox
- watch commit graphs animate as branches split/merge/rebase
- solve LeetCode-style Git missions with hints, score, XP, and streaks
- track progress with username-based profiles and leaderboard
- challenge friends using shareable mission links

The purpose is simple:  
help beginners build intuition fast, and help experienced devs sharpen advanced workflows through simulation.

Built with Next.js + TypeScript + Framer Motion, deployed on GitHub Pages.

If Git ever felt confusing, this project is for you.

Try it here:  
https://sureshkonar.github.io/GitVisualizer/
